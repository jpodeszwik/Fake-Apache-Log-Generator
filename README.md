# Fake Apache Log Generator

## Instalation (ubuntu)
* ```sudo apt-get install build-essential python-dev python-pip python-virtualenv```
* ```virtualenv virt_env```
* ```source virt_env/bin/activate```
* ```pip install -r requirements.txt```

## Usage
* ```./run.sh <hostname> <port>```
